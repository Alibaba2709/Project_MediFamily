# Medifamily

Medifamily is a full-stack web application designed to help families manage healthcare information in one place.

## Features
- User authentication
- Appointment management
- Medication tracking
- Medical documents
- Reminders
- Family profiles

## Tech Stack
- React
- Next.js
- Node.js
- MongoDB
- Git/GitHub
- Vercel
